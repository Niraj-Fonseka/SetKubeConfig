# setkubecontext

Switch between Kubernetes contexts easily

## Installation 
```
go get -u github.com/Niraj-Fonseka/setkubecontext
```
or download the binary from from one of the releases


## Running the app

### MacOS / Linux 
```
cp $GOPATH/bin/setkubecontext /usr/local/bin
```

In the terminal run 
```
setkubecontext
```
